# API Reference Overview

APIリファレンスドキュメントへようこそ。

## このセクションについて

このセクションでは、プロジェクトで使用できるAPI、関数、フック、ユーティリティのリファレンスを提供します：

- Reactコンポーネント API
- カスタムフック
- ユーティリティ関数
- 型定義
- 設定オプション

## 主要なAPI

### コンポーネント

- `Button` - ボタンコンポーネント
- `Card` - カードコンポーネント
- `ArticleCard` - 記事カードコンポーネント
- `ProjectCard` - プロジェクトカードコンポーネント
- `DocsCategoryCard` - ドキュメントカテゴリーカードコンポーネント

### データ構造

- `articles.json` - 記事データ
- `projects.json` - プロジェクトデータ
- `profile.json` - プロファイルデータ
- `docsCategories.json` - ドキュメントカテゴリーデータ
