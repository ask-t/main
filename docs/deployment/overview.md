## クイックデプロイ（Vercel）

Vercelは最も簡単なデプロイ方法です：

1. Vercelアカウントにサインイン
2. GitHubリポジトリをインポート
3. ビルド設定を確認（自動検出）
4. デプロイボタンをクリック

詳細な手順は [DEPLOYMENT_GUIDE.md](/DEPLOYMENT_GUIDE.md) を参照してください。
