# ドキュメント

ask-tのポートフォリオサイトに関するドキュメントへようこそ。

## サイトの概要

このポートフォリオサイトは以下の技術で構築されています：

- **フレームワーク**: Docusaurus
- **言語**: TypeScript, React
- **スタイリング**: Tailwind CSS
- **デプロイ**: Vercel

## 開発環境のセットアップ

### 必要な環境

- [Node.js](https://nodejs.org/en/download/) バージョン 18.0以上
- npm または yarn

### 開発サーバーの起動

```bash
npm install
npm start
```

開発サーバーが起動すると、http://localhost:3000 でサイトを確認できます。

### ビルド

```bash
npm run build
```

本番環境用のビルドが `build/` ディレクトリに生成されます。
